# client 使用 vue-cli3.0 + TS 搭建的博客项目

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```


### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

```
yarn run test:unit
```
